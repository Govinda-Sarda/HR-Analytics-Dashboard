# HR-Analytics-Dashboard
• Designed an HR analytics dashboard analyzing employee data across 3 departments (HR, RND, Sales) with 1,000+
records to identify attrition trends.
• Visualized 10+ KPIs including attrition rate, age distribution, salary slabs, and tenure using interactive Power BI
dashboards.
• Implemented dynamic filters and DAX calculations for multi-level drill-down analysis, supporting data-driven
workforce planning decisions.
